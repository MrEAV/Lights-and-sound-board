# Lights-and-sound-board
Lights and sound board for delayed speech, non verbal etc. customizable audio file, web app.
The idea is to take a ESP32 and flash it with a custom firmware that will allow arcade light up buttons to be correlated to an audio file.
The goal is to have a customizable sound board that will allow you to press a button and a speech pattern is played, for example (i need to go potty), or even a 1-10 counting sequence.
more or less it should be web app interfacable, allow you to drop various mp3 files to the storage and assign the specified button.
can be modified how deemed fit. 
i will update as i go.

materials:
MT3608 boost converter 
SanDisk 32GB microSD 
30mm LED arcade buttons x 10 
18650 battery holder 
TP4056 charging module 
DFPlayer Mini 
3W 8Ω speaker
